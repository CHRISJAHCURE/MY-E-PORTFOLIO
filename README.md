# MY-E-PORTFOLIO
MY PERSONAL E-PORTFOLIO BUILT WITH HTML, CSS AND JAVASCRIPT
It showcases my skills, services, and provides a contact form to reach me.

---

## Features
- Home/About Me section with my personal details
- Services section with list of skills and services I offer
- Contact form with JavaScript functionality
- Responsive and professional design
- Profile picture included

---

## Usage
1. Open index.html in a web browser
2. Navigate through Home, Services, and Contact sections
3. Click the "Send Message" button to test the contact form functionality
